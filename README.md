Number Formatter
=========

A small library that adds commas to numbers

## Installation

  `npm install @ambichetan/number-formatter`

## Usage

    var numFormatter = require('@ambichetan/number-formatter');

    var formattedNum = numFormatter(35666);


  Output should be `35,666`


## Tests

  `npm test`
